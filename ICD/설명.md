### Frame 설명
- STX : 시작 바이트 (0xAA) -> 고정!
- CMD : 명령코드
- LEN : 페이로드 길이
- PAYLOAD : 데이터
- CHK : 체크 (XOR)
- ETX : 종료바이트 (0x55) -> 고정!

<br>

### CHK 계산방법 (XOR) -> PAYLOAD (O)
1. 초기값: 0x00
2. CMD ^ LEN
3. (2) ^ PAYLOAD(1)
4. (3) ^ PAYLOAD(2)
5. PAYLOAD 끝 값까지 계속 XOR 연산
6. 연산 최종값 -> CHK!

<br>

### CHK 계산방법 (XOR) -> PAYLOAD (X)
1. 초기값: 0x00
2. CMD ^ LEN
3. 2번째 값 -> CHK!
